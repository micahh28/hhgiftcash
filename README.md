### 上手指南

本项目需要配合，服务端： https://github.com/micahh28/HHGiftCashApi

###### **安装步骤**

```bash
# install
npm install

# build dist files
npm run serve

# serve examples at localhost:8080
npm run build

# lint 
npm run lint
```
